
# Shop Manager

## Commands

**/shop-manager add**
    - Parameters: <name, description, price, stock>
    - Description: Add an item to the shop

**/shop-manager remove**
    - Parameters: <name>
    - Description: Remove an item from the shop

**/shop-manager edit**
    - Parameters: <name> [description, price, stock]
    - Description: Edit an item in the shop

## Permissions
- Required the `Server Staff` role
